# 🛍️ Store Sales Analysis Dashboard (Excel)

## 📌 Project Overview

The **Store Sales Analysis Dashboard** is an interactive Microsoft Excel dashboard designed to provide valuable insights into sales performance, customer demographics, order trends, and sales channels. Built using Excel's powerful data visualization features, this dashboard enables businesses to monitor key metrics, identify high-performing regions, and make informed business decisions.

The dashboard combines KPIs, Pivot Tables, Pivot Charts, Slicers, and interactive filters to transform raw sales data into meaningful business intelligence.

## 🎯 Business Objective

Retail businesses generate thousands of transactions across different states, sales channels, and customer segments. Analyzing this data manually is challenging and time-consuming.

This dashboard helps answer key business questions such as:

* Which months generated the highest sales?
* What is the relationship between orders and revenue?
* Which gender contributes the most sales?
* Which age group places the most orders?
* Which states generate the highest revenue?
* Which sales channels perform the best?
* What is the overall order delivery status?

# 📈 Key Insights

* Women contribute **64%** of total sales, while men contribute **36%**.
* Adult customers account for the highest number of orders.
* Maharashtra records the highest sales among all states.
* Amazon is the leading sales channel, followed by Myntra and Flipkart.
* Approximately **92%** of orders are successfully delivered.
* Sales peak during March, indicating strong seasonal demand.
* Order volume and revenue show a closely correlated trend throughout the year.
